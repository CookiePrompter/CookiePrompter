Motomo tracker

Her er Motomo trackeren. Kig i source for at se, hvordan den er sat op.

Den virker kun, hvis du sætter trackerUrl samt siteId i initialiseringsscriptet

Hvis domain ikke bliver sat, sætter CookiePrompter det til *.document.href.location

Klik herunder for at slette de cookies, der bliver sat.


                CookiePrompter.init({
                    readMoreUrl: 'readmore.html',
                    enableLog: false,
                    trackers: [{
                        name: MotomoTracker,
                        config: {
                            trackerUrl: "https://127.0.0.1:4433", // https://xxx.motomo.cloud
                            siteId: 1,
                            domain: "",  // use *.domain.com for all subdomains 
                        }
                }]
        });
